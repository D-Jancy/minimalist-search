重磅消息 ｜ Aqqle 宣布旗下所有操作系统、软件开源
Created 04-01-2020| Updated 04-10-2020|Tech
Word count:1.4k|Reading time: 4 min
| Post View:|Comments:

几天前，Aqqle 便释出了想要积极加入开源社区的信号，在重金招募 Open Source Program Engineer。众所周知，作为全球最大的封闭生态环境（软硬件），Aqqle旗下的大部分产品都属于闭源。闭源自然有闭源的好处：从未被在其他设备上安装过的aaiOS是当之无愧的最持久系统，aiPhone 6S 在安装 aaiOS 13 后甚至还能再战半年，而相比许多同时代的安桌机型，早已成为历史的遗物。但是，这样的环境也有弊端，相对高昂的价格使 Aqqle 失去了大批发展中国家用户，作为一家盈利的商业公司，这样的做法未免有失其准则。

近些天，尽管 Aqqle 的保密措施一向严格，但被称作科技节春晚的苹果核年度发布会被泄露的越来越多，甚至还有在 aiPhone 7 上跑安桌的现象产生，而且其最大的竞争对手微软近期在开源方面也有很大动作，自从收购了 GitHub ，微软拥抱开源的想法越来越强烈，可是，苹果核好像有些坐不住了，提前在2020年4月1日释出了惊天动地泣鬼神的重大消息—— Aqqle 全系软硬件转向开源社区。

在其官网上， Aqqle 毫不掩饰的发布了自己拥抱开源的积极态度，同时上线了 GitHub。对于 Aqqle 突然转向开源社区，官方给出的评论只有八个大字：生死看淡 不服来干。相信很多人和我们一样对 Aqqle 突然拥抱开源有很大的疑惑，今天，我们有幸采访到了 Aqqle 内部高级工程师—— Cyberduck。Cyberduck是苹果内务处mecOS 部门的高级架构工程狮，长期负责处理 mecOS 每天遇到的成千上万的bug，为mecOS转型bugOS做出了巨大贡献，今天，我们就将对 Aqqle 拥抱开源的问题采访一下他：

Q: 您好！今天突然得知 Aqqle 旗下的产品全线转入开源阵营，想请问一下您：苹果之前对待开源的态度是怎么样的？

Cyberduck: 呢個啊，開源嘛，我嘅老朋友鮑爾默曾經講過，開源就系知識產權嘅癌症，呢個癌症嚟得急、來勢猛，我哋對開源嘅態度也系咁嘅。我哋公司內部采攞了嚴格嘅保密措施，把源代碼全部私有化。（CHS: 这个啊，开源嘛，我的老朋友鲍尔默曾经讲过，开源就是知识产权的癌症，这个癌症来得急、来势猛，我们对开源的态度也是这样的。我们公司内部采取了严格的保密措施，把源代码全部私有化）

Q: 那是什么促使 Aqqle 转向了开源呢？

Cyberduck: 呢依舊要講到鮑爾默了。呢小子自己嘴上咁講，卻仲系讓微軟全身心投入了開源。我哋唔想喺呢場戰役中失去利益，所以只好加入開源社區。(CHS: 这依旧要讲到鲍尔默了。这小子自己嘴上这样讲，却还是让微软全身心投入了开源。我们不想在这场战役中失去利益，所以只好加入开源社区。)

Q: 所以从一定程度上讲，Aqqle 加入开源是迫不得已的？

Cyberduck: 也唔全系。隨着mecOS、aiOS Bug越嚟越多，開源化嘅潮流越嚟越猛，我哋發現單靠一個唔對外開放嘅團隊嚟處理呢些問題實喺系太困難了。開源背後有一成個社區嚟驅動，而我哋過去只有一個團隊，而且團隊還要嚴格對外保密。(CHS: 也不全是。随着 mecOS、aiOS Bug 越来越多，开源化的潮流越来越猛，我们发现单靠一个不对外开放的团队来处理这些问题实在是太困难了。开源背后有一整个社区来驱动，而我们过去只有一个团队，而且团队还要严格对外保密。)

Q: 您认为对于苹果来说，加入开源社区是一件好事还是坏事？

Cyberduck: 我認為仲系一件好事嘅，第一系冇了保密嘅負擔，同時也會有更多嘅人嚟參與測試，參與Bug修復工作。近嚟mecOS嘅bug真系越嚟越多了，我覺得有一些無力回天了。(CHS: 我认为还是一件好事的，第一是没有了保密的负担，同时也会有更多的人来参与测试，参与 Bug 修复工作。近来 mecOS 的 bug 真是越来越多了，我觉得有一些无力回天了。)

Q: 对于开源社区呢？

Cyberduck: 呢個唔太好講。蘋果嘅加入對開源社區嚟講勢必系一大沖擊，但究竟系好系壞還很難下定論。(CHS: 这个不太好讲。苹果的加入对开源社区来讲势必是一大冲击，但究竟是好是坏还很难下定论。)

Q: 您是否考虑跳槽到 WeBlog 工作？

Cyberduck: 真嘅嘛！！呢個系我嘅畢生追求，能加入 WeBlog 工作實喺系我嘅榮幸，我嘅辭職信已經喺我嘅草稿箱里存了太久了。 Bingo～佢發出去了！(CHS: 真的嘛！！这个是我的毕生追求，能加入 WeBlog 工作实在是我的荣幸，我的辞职信已经在我的草稿箱里存了太久了。 Bingo～他发出去了！)

你对苹果核加入开源社区有怎样的看法？欢迎加入讨论～
